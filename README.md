# ToDo List with ReactJS and ReduxJS

> ToDo List with ReactJS and ReduxJS

### Setup Instructions

> 1. Fork this repo
> 1. Clone your forked copy of this repo
>    - `git clone https://github.com/aliawilkinson/todolist_redux.git`
> 1. Change directory into the newly cloned repo
>    - `cd todolist_redux`
> 1. Install dependencies 
>    - `npm install`
> 1. Start dev server
>    - `npm start`
